# Web3 NFT Wallet Viewer

This is a simple Web3 project.  
It helps you check what NFTs a wallet owns using basic blockchain logic.

This project is made for beginners who want to learn how NFTs and wallet data work.

## What This Project Does
- Read wallet address
- Fetch NFT list from API
- Count NFTs
- Show NFT names in console
- Save NFT data to file

## Tools Used
- Node.js
- Simple JavaScript
- Public NFT API (mock)
- Local JSON file for storage

## How To Install
1. Install Node.js
2. Download this repository
3. Open terminal in project folder
4. Run:
   npm install
5. Start the app:
   node app.js

## Folder Structure
- app.js = main file
- config.js = project settings
- nft.js = NFT logic
- api.js = API calls
- saver.js = file saving
- data/ = saved NFT data
- logs/ = log files

## Example Use
- Set wallet address
- App fetches NFTs
- App shows total NFT count
- App saves NFT list to file
